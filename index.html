Naming Conventions

Rules for naming files and paths

  • all lowercase - different operating systems treat capitilization differently (Good Example: ten-tree-logo.svg Bad Example: Today-in-History.html)
  • No spaces - URLs cannot have spaces in them (Good Example: chocolate-cake-recipe.html Bad Example: chocolate cake recipe.html)
  • Only letters, numbers, and dashes (-) - URLs use a limited set of characters (Good Example: west-coast-art-2021.html Bad Example: red_porche.jpg

Commit messages

  • minimum 3 words and 10 characters
  • Proper spelling and grammer
  • Capital starting letter, no period
  • Must start with an imperative verb: Create, Fix, Add, Solve, etc
  • Pretend every commit starts with the phrase: This commit will...
